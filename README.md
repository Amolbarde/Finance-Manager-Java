# 💰 Finance Manager System (Java + PostgreSQL)

A desktop-based Finance Manager application built using Java Swing and PostgreSQL.  
It allows users to register, login, and view their personal details.

---

## 🚀 Features

- 🔐 User Registration with validation
- 🔑 Secure Login System
- 📊 Dashboard with user information
- 🗄️ PostgreSQL Database Integration
- ❌ Duplicate username prevention

---

## 🛠️ Tech Stack

- Java (Swing GUI)
- JDBC (Database Connectivity)
- PostgreSQL
- Eclipse IDE

---

## ⚙️ Database Setup

1. Open PostgreSQL
2. Create database:
   CREATE DATABASE register;
3. Run:
   database.sql
   

---

## ▶️ Run Project

### Option 1: Eclipse
- Import project
- Run `LoginFrame.java`

### Option 2: JAR
java -jar FinanceManager.jar
